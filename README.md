# Step of use Explorer

## Getting Started

1. **Install dependencies**
	```bash
	npm install
	```

2. **Start the development server**
	```bash
	npm run dev
	```

3. **Open in browser**
	Navigate to [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal).
